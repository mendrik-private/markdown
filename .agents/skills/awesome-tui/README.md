# awesome-tui

Agentic skill for building polished, keyboard-first terminal interfaces inspired by `sqv`.

## Install

```bash
npx skills add mendrik-private/awesome-tui
```

Or install from the SSH URL:

```bash
npx skills add git@github.com:mendrik-private/awesome-tui.git
```

## What it enforces

- Dogmatic warm TUI theme with sacred accent color
- Truthful help and status surfaces
- Tabs, panel split navigation, and focus states
- Large-data scrolling and grid behavior
- Mouse interactions with keyboard equivalents
- Keyboard shortcut conventions for terminal apps
- Popup, toast, confirmation, and command palette patterns

The skill is defined in [`SKILL.md`](SKILL.md), which is discoverable by the open agent skills ecosystem and compatible with `skills.sh`.
